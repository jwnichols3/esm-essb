How do I create a suppression rule from the web interface?



1. Go the the EEB portal and login. You should be able to login with your Windows username/password.

2. Navigate to the Data Map page.

3. Click on the "Add New Record" link.

4. Fill in the following form with the pertinent information. You can hover over the "Help" link to get more information.
