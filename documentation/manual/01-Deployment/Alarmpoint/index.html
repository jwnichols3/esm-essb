Setting up the Alarmpoint Integration client.



1. Make sure that the Service Center integration binary is in the Alarmpoint Java Client directory.

The name of this binary is scgateway_ws_client.exe.
2. Change the "APAgent.xml" file under the "etc" directory in the Alarmpoint Java Client directory.

There will be a line that looks like the following:

eebUpdateCommand.append ( "cmd /c c:\\APAgent\\scgateway_ws_client.exe -mode=qa -command=updateIncident -incidentID=" );


Make sure that the only one of the following lines ( dev, qa, ppc, prod) is uncommented. There are two places in the APAgent.xml file that needs to be changed.
3. Restart the Alarmpoint Java Client.
